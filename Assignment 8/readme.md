# 💰 Personal Finance Tracker

A modern and responsive **Personal Finance Tracker** built using **HTML, CSS, and Vanilla JavaScript**. This project helps users manage their income and expenses efficiently with a clean dashboard, transaction management, and local storage support.

---

## 🚀 Features

### 🔐 Authentication
- User Signup
- User Login
- Logout
- Local Storage Authentication

### 📊 Dashboard
- Welcome Section
- Total Balance
- Total Income
- Total Expense
- Total Savings
- Recent Transactions (Last 5)
- Top Spending Categories

### 💳 Transaction Management
- Add Transaction
- Edit Transaction
- Delete Transaction
- Dynamic Categories
- Income & Expense Support

### 📄 Transaction Page
- View All Transactions
- Search Transactions
- Filter by Category
- Filter by Type
- Sort by Date & Amount
- Pagination

### 📱 Responsive Design
- Desktop Layout
- Tablet Responsive
- Mobile Responsive
- Collapsible Sidebar

### 💾 Storage
- Local Storage Persistence
- User-wise Transaction Data
